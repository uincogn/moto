# 🔧 Scripts KM$

Esta pasta contém scripts utilitários para desenvolvimento, teste e deployment.

## 📁 Scripts Disponíveis

### 🧪 Teste
- **`test_backend.sh`** - Script para testar todas as rotas do backend API

### 🚀 Deploy (Futuros)
- **`deploy.sh`** - Script de deployment automatizado
- **`build.sh`** - Script de build para produção
- **`setup.sh`** - Script de configuração inicial

## 📋 Como Usar

### Testar Backend
```bash
# Dar permissão de execução
chmod +x scripts/test_backend.sh

# Executar teste
./scripts/test_backend.sh
```

**Pré-requisitos:**
- Backend rodando na porta 3000
- curl instalado
- jq instalado (para formatação JSON)
